# Headless WordPress Blog Site

A headless wordpress blog using Gatsby to build a decoupled frontend with Framer Motion as the the animation library to create a stylish landing page. Wordpress is used as the CMS which allows the owner of the blog to update the content freely in the wordpress dashboard.
